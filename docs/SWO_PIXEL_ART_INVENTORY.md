## Asset Sourcing Strategy

### When to use free asset libraries
- **Sanctuary UI** (shop, inventory, HUD): Start with CraftPix RPG UI (#20) or Kenney UI (#21), dark-palette swap
- **Companion sprites**: Pick from Dungeon Crawl (#1), 50+ Monsters (#2), or NightBorne/Necromancer (#3/#4)
- **World map base**: CraftPix cosmic backgrounds (#14) + Puny World tiles (#28) recolored
- **Icons/badges**: 496 RPG Icons (#18) cover most needs; Sci-Fi Icons (#19) for cosmic flavor
- **Effects**: Pixel Art Spells (#23) in B&W + recolor; Gothicvania Magic (#24) for dark combat VFX

### When to generate pixel art
- **Unique SWO branding**: Logo, token icons, faction emblems — these must be original
- **Companion identity art**: Named companions need custom sprites (not generic monsters)
- **Story-specific assets**: Quest-specific items, unique NPCs, narrative scenes
- **Anything that becomes a selling point**: If it differentiates SWO, don't use stock

### Keeping style consistency across mixed sources
1. **Define a master palette**: Pick 16-32 colors (dark purples, cosmic blues, gold accents, deep reds). All assets get palette-mapped to this.
2. **Standardize pixel density**: Pick one canonical size per context (e.g., 32x32 for map tiles, 64x64 for companions, 128x128 for portraits). Rescale everything to these.
3. **Outline convention**: Decide on dark outlines vs. no outlines. Most dark fantasy packs use dark outlines — stick with that.
4. **Post-processing pass**: Run all sourced assets through a batch palette-swap + outline-normalization script before shipping. This is the single most important step for coherence.
5. **Asset provenance tracking**: Tag each asset with source + license in a manifest file so attribution obligations are met and origins are traceable.

### Priority sources (ranked)
1. **CraftPix freebies** — Best dark theme fit, royalty-free, no attribution
2. **OpenGameArt CC0** — Largest CC0 collection, Dungeon Crawl pack is exceptional
3. **itch.io NYP** — Best individual character quality (ansimuz, CreativeKind, aamatniekss)
4. **Kenney CC0** — Best for structural/UI foundations, needs recoloring
5. **Freepik** — Skip unless premium; attribution requirement is friction
