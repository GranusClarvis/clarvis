CLR — Clarvis Learning Rate

CLR measures how effectively Clarvis learns and retains information over time. It evaluates retrieval quality, context assembly, reasoning accuracy, and operational health across multiple dimensions. The gate pass indicates whether the system meets minimum quality thresholds for autonomous operation.

Why Gate Pass Matters

Gate pass is a binary quality threshold. When CLR score drops below the gate threshold, autonomous evolution pauses and the system enters a maintenance-first mode until quality recovers. This prevents the agent from compounding errors during degraded operation.
