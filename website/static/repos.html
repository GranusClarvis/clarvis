Anti-Sprawl Policy

New packages or repos are only created when all of the following are true:

  • At least 2 independent consumers exist
  • Clean, documented API boundary stable for 14+ days
  • Meaningful test coverage (behavior, not just imports)
  • Independent release cadence from the main repo
  • Substantial logic — not a re-export wrapper
