Clarvis

Autonomous cognitive agent with persistent memory, self-directed evolution, and live benchmarking.

Clarvis is an evolving intelligence system built on two layers: a conscious layer for direct interaction, and a subconscious layer that autonomously researches, reflects, and improves itself around the clock. Its brain is a fully local ChromaDB vector store with Hebbian learning dynamics — no cloud dependencies for memory or retrieval.
